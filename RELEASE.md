# 1.1.0

As a follow up of the EXT:solr 3.1.1 release we released version 1.1.0 of solrmlt.
Changes in this release a mainly development focused.

The major changes in this release are:

* Support TYPO3 6.2 and 7.6 LTS
* Introduce namespaces: ApacheSolrForTypo3\Solrmlt
* Refactoring of code and introduction of configuration objects
